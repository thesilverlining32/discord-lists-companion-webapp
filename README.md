# My List App

A web application to manage lists of ideas, games, books, movies, and TV shows with Discord authentication.

## Setup

1. Clone the repository
2. Install dependencies for the backend and frontend:
	```bash
	cd backend
	npm install
	cd ../frontend
	npm install
	```
